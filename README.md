# STRIKE OS Website

![STRIKE OS](https://strikeos.site/assets/logo/logo_S.webp)

**Live Demo:** [https://strikeos.site/](https://strikeos.site/)

This repository contains the source code for the **STRIKE OS** landing page. STRIKE OS is a highly-tuned Windows OS engineered for maximum framerates, zero bloatware, and ultra-low latency, built specifically for competitive gamers and power users who demand peak hardware performance.

## 🌟 Features

*   **Modern UI/UX:** A clean, modern, and high-contrast design featuring a Deep Black and Neon Purple color scheme.
*   **Liquid Glass Design:** Premium "liquid glass" (glassmorphism) effects applied to cards and components for a sleek aesthetic.
*   **Responsive Layout:** Fully optimized for both desktop and mobile devices.
*   **Multi-language Support:** Built-in support for English and Vietnamese (VN) translations.
*   **Smooth Animations:** Incorporates smooth scrolling, reveal animations, and a neon scroll progress bar.
*   **Parallax Background:** Interactive particle background for an immersive visual experience.

## 🚀 Sections

1.  **Hero:** Introduction to STRIKE OS with a call-to-action to join the Discord community.
2.  **Performance:** Visual benchmarks showcasing the performance benefits over Stock Windows (Coming Soon).
3.  **Services:**
    *   **Strike OS - Window Custom:** A custom Windows build optimized for performance and stability.
    *   **Window Optimization:** Deep system tuning, including registry, network, and latency reduction.
    *   **PC Building:** Professional consulting and PC assembly tailored to your needs and budget.
    *   **Tuning BIOS:** Advanced BIOS tuning including EXPO/DOCP, PBO, Curve Optimizer, and latency tweaks (Specialized for AMD Ryzen X3D CPUs).
4.  **Feedback:** A dynamic carousel displaying user reviews and testimonials.

## 🛠️ Technology Stack

This project is built using vanilla web technologies, ensuring maximum performance and a lightweight footprint:

*   **HTML5:** Semantic structure.
*   **CSS3:** Custom styling, Flexbox/Grid layouts, CSS variables, and glassmorphism effects (`backdrop-filter`).
*   **JavaScript (Vanilla):** DOM manipulation, event handling, multi-language switching, and scroll animations.
*   **FontAwesome:** For scalable vector icons.
*   **Google Fonts:** Utilizing the 'Montserrat' font family for excellent readability and a modern look.
